#项目思路:

*1.用户进入预约页面，验证通过之后点击提交按钮提交数据到数据库<br>  


*2.后台预约界面从数据库获取数据添加到后台页面<br>  


*3.考虑到服务器问题这里用json代替实现简单的数据交互<br>   


####工作流程   


*1.cd desktop->mkdir demo-xdf->cd demo-xdf->npm init(创建page.json文件)<br>  


*2.后台预约界面从数据库获取数据添加到后台页面<br>  


*3.考虑到服务器问题这里用json代替实现简单的数据交互<br>
